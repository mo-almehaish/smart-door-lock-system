Smart Door Lock System 🔐

A smart door lock system project using Arduino Uno and embedded systems concepts.

Features

* Password authentication
* LCD display
* Electronic lock control
* Servo motor operation

Components Used

* Arduino Uno
* LCD Display
* Keypad
* Servo Motor
* Jumper Wires
* Power Supply
